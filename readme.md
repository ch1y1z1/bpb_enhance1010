# 背包乱斗摆盘测试辅助工具

这个项目可以帮助您方便快捷地在游戏中测试摆盘与对战强度。

重要声明：该辅助工具仅支持在自定义规则模式测试使用。由于技术限制，所有需要联网的模式（包括排位、休闲、派对）均不能使用辅助工具。其他玩家匹配不到自定义规则模式中摆出的阵型。

## 安装

在 Release 中下载已经编译好的二进制文件（bpb_helper.exe）。运行下载好的文件。

![安装软件截图](./pics/image.png)

选择游戏路径（如果安装在默认路径可以跳过这一步）。

点击“应用”按钮。如果修改成功，则可以直接删除下载的文件。

如果需要复原游戏文件，请通过 Steam -> 验证游戏完整性。

## 使用方法

经过修改后的游戏提供两个功能：

1. 快捷键“J”：一键跳转至 18 回合，直接打决赛局。
2. 在物品图鉴页面右键点击物品，可以直接在仓库中生成该物品。

您的阵型同样会被保存在历史记录中，可以后续进行查看。
